Introducing the testing frameworks

  • JUnit 5
  • Spek

JUnit 5 - Introduction

  • Just like you are used to from Java
  • JUnit 5 is modularized
  • Fixes issues from earlier versions or made features easier to use
  • RunWith replaced by ExtendWith
  • Works like a charm with Spring and other frameworks

Spek - Introduction

  • DSL for testing
  • May be familiar if you use JS (Ava, Jest etc.)
  • Uses JUnit 5 as an engine
  • Spring and other framworks mostly tedious to test
